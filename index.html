Je suis développeuse Python, avec une spécialisation en développement backend, traitement de données et intégration frontend.
Au cours de ma formation, j’ai développé un large éventail de compétences en programmation, avec un accent particulier sur l’écosystème Python et le développement web. J’ai travaillé sur des projets concrets tels que des pipelines ETL, des applications web complètes avec Django et Flask, ainsi que des API REST performantes.
J’ai également exploré une diversité de technologies — HTML, CSS, JavaScript, PHP, Bash, Node.js, Visual Basic, C# — à travers des projets personnels et professionnels, renforçant ainsi ma polyvalence technique.

Voici un aperçu des projets et des domaines techniques abordés :


Pipelines de données (ETL) : Conception de processus d’extraction, de transformation et de chargement de données depuis des plateformes en ligne vers des fichiers CSV structurés, en vue d’analyses automatisées.

Programmation orientée objet et design patterns : Rédaction de code Python clair et maintenable selon les standards PEP8, en appliquant des modèles de conception (MVC, Singleton...) pour structurer les projets de manière modulaire.

Développement web fullstack :
Frontend : Création d’interfaces réactives avec HTML, CSS et JavaScript, intégrées avec des APIs RESTful.
Backend avec Django : Développement d’applications web complètes incluant inscription, authentification, gestion de contenu (flux, commentaires, abonnements) et interactions avec des bases de données relationnelles.
API REST sécurisées : Mise en place d’API robustes avec Django REST Framework, gestion de l’authentification via OAuth2, et tests des endpoints avec Postman.



Méthodologies avancées :
Optimisation algorithmique : Résolution de problèmes complexes avec analyse Big O et implémentation d’algorithmes efficaces (brute-force, dynamiques…).
TDD et debug avec Flask : Utilisation du Test-Driven Development pour assurer la qualité du code dans des projets Flask, avec une attention portée sur la gestion des erreurs.
Bases de données & sécurité backend : Conception de schémas SQL robustes, gestion sécurisée de l’authentification et des flux de données côté serveur.
Outils DevOps & bonnes pratiques :
Refactorisation pour améliorer la performance et l’évolutivité. Intégration d’outils comme Sentry (monitoring), ReadTheDocs (documentation) et GitHub Actions/GitLab CI pour l’automatisation des tests et du déploiement (CI/CD).



Ma philosophie de développeuse :

Pour moi, être développeuse, c’est avant tout une démarche d’apprentissage continu. J’aime comprendre en profondeur, expérimenter, et approfondir ma compréhension des concepts techniques afin de proposer des solutions pertinentes..
Mon approche est centrée sur la recherche de solutions simples et robustes à des problématiques concrètes, tout en restant ouverte aux nouvelles technologies et aux bonnes pratiques du métier.
Je m’attache à produire un code clair, fiable et bien documenté, en gardant toujours une vision pragmatique et orientée vers l'efficacité.

Project BooksOnlineScraper

This script is designed to scrape detailed information about books from an online bookstore. The script navigates through various book categories, extracts relevant data for each book, and saves the data to CSV files. Additionally, it downloads and saves images associated with each book.

Project OCMovies

This web application allows users to discover movies based on their genre and IMDB rating. The application retrieves data from an API to display movies and their details in an interactive and responsive interface.

Project LitReview

P9 LitReview est une application web permettant à une communauté d'utilisateurs de publier des critiques de livres, consulter des avis et solliciter des critiques sur des œuvres spécifiques.

Project API Restful

ApiRestful est une API REST complète pour la gestion collaborative de projets logiciels. Elle permet aux utilisateurs de créer des projets, de collaborer via des issues et des commentaires, le tout en respectant une hiérarchie de permissions rigoureuse (auteurs, contributeurs, administrateurs).
